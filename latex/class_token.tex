\doxysubsection{Description détaillée}
Classe \mbox{\hyperlink{class_token}{Token}}

Cette classe représente un jeton trouvé dans une chaîne de caractères spécifiée.

Un jeton est une suite de caractères dans un programme source qui correspond au motif d\textquotesingle{}un symbole lexical et qui est analysé par le \mbox{\hyperlink{class_tokenizer}{Tokenizer}} comme une instance de ce jeton lexical.

\end{DoxySeeAlso}
