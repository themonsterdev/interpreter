◆ GetStringToken()

string StreamTokenizer::GetStringToken ( )

Si le jeton actuel est un jeton de type mot, ce champ contient une chaîne donnant les caractères du jeton de type mot. Si le jeton actuel est un jeton de chaîne entre guillemets, ce champ contient le corps de la chaîne.

Le jeton actuel est un mot lorsque la valeur du champ ttype est TT_WORD. Le jeton actuel est un jeton de chaîne entre guillemets lorsque la valeur du champ ttype est un caractère entre guillemets.
